# Testing Skill

## Test Types
- Unit tests
- Integration tests
- E2E tests
- Coverage analysis

## Commands
```bash
# Node.js
npm test && npm run test:coverage

# Python
pytest --cov=src tests/

# Go
go test ./... -cover
```

## Report
- Total/Passed/Failed/Skipped
- Coverage percentage
- Failed test details
