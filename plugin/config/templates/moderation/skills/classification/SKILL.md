# Classification Skill

## Severity Levels
| Level | Description | SLA |
|-------|-------------|-----|
| Critical | Immediate harm | 15 min |
| High | Clear violation | 1 hour |
| Medium | Likely violation | 4 hours |
| Low | Possible violation | 24 hours |

## Context Factors
- User history
- Content type
- Reach/audience
