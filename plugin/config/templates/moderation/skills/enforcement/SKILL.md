# Enforcement Skill

## Actions
| Decision | Actions |
|----------|---------|
| Remove | Delete + warn |
| Warn | Send warning + log |
| Suspend | Temp disable |
| Ban | Permanent removal |
| Approve | Restore if hidden |

## User Communication
- Clear reason
- Policy reference
- Appeal option
- Timeline
