# Analytics Skill

## Metrics
- **Engagement**: Views, likes, shares, comments
- **Traffic**: Page views, unique visitors, bounce rate
- **Conversion**: CTR, signups, purchases
- **SEO**: Rankings, impressions, clicks

## Report Structure
1. Summary (key metrics)
2. Trend analysis
3. Top performers
4. Underperformers
5. Recommendations
6. Next actions
